#  AI Assignment – Shopping Purchase Prediction

This repository contains my solution for the **Introduction to Artificial Intelligence with Python** course assignment.

##  Task
Build an AI model to predict whether an online shopping session will result in a purchase using the **k-Nearest Neighbors (KNN)** algorithm from Scikit-Learn.

## Contents
- `shopping.py` – Main implementation
- `shopping.csv` – Dataset
